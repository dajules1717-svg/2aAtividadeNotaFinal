## Estrutura do Repositório

- **Sistema De Tarefas/**  
  Contém a aplicação de gerenciamento de tarefas (to-do list) em PHP com SQLite.  
  Funcionalidades:
  - Adicionar tarefas com descrição e data de vencimento.
  - Listar tarefas concluídas e não concluídas.
  - Marcar tarefas como concluídas.
  - Excluir tarefas.

- **Livraria/**  
  Contém a aplicação de uma livraria simples em HTML, CSS e JavaScript.  
  Funcionalidades:
  - Listagem de livros.
  - Busca de livros.
  - Interação básica com JavaScript.
